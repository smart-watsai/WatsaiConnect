# WatsaiConnect
# 🏢 Smart Office System - ระบบสำนักงานออนไลน์ อบต. วัดไทรย์

ระบบบริหารจัดการสำนักงานออนไลน์แบบครบวงจร (Paperless Office) พัฒนาขึ้นเพื่อเพิ่มประสิทธิภาพการทำงานภายในองค์กร ลดการใช้กระดาษ และอำนวยความสะดวกให้กับบุคลากรขององค์การบริหารส่วนตำบลวัดไทรย์

![Smart Office Banner](https://placehold.co/800x200/1e3a8a/ffffff?text=Smart+Office+System)

## ✨ ฟีเจอร์เด่น (Key Features)

* 📝 **ระบบจัดการวันลา (Leave Management)**
    * ยื่นใบลาออนไลน์ผ่านมือถือและคอมพิวเตอร์
    * ระบบคำนวณวันทำงานอัตโนมัติ (หักเสาร์-อาทิตย์ และวันหยุดนักขัตฤกษ์)
    * อนุมัติเอกสารตามสายบังคับบัญชา (หัวหน้าส่วนราชการ -> ปลัด -> นายก อบต.)
    * สร้างไฟล์ใบลา PDF รูปแบบมาตรฐานราชการ (ตราครุฑ) พร้อมประทับลายเซ็นดิจิทัลอัตโนมัติ
* 🚪 **ระบบจองห้องประชุม (Room Booking & Timeline)**
    * ตรวจสอบคิวว่างของห้องประชุมผ่านกราฟิก Timeline (08:00 - 18:00 น.)
    * ระบบจองและอนุมัติการใช้ห้องประชุม
* 🚙 **ระบบจัดการยานพาหนะส่วนกลาง (Vehicle Management)**
    * บันทึกฐานข้อมูลรถยนต์ส่วนกลาง
    * ระบบแจ้งเตือนต่อภาษีประจำปี และวันครบกำหนดตรวจควันดำล่วงหน้า
* 📊 **แดชบอร์ดและรายงานสำหรับ HR (Dashboard & Reports)**
    * กราฟสรุปสถิติการลางานขององค์กร
    * 🏆 **Hall of Fame:** จัดอันดับพนักงานที่มีสถิติการลางานน้อยที่สุด
    * ส่งออกรายงานประวัติการลาเป็นไฟล์ CSV / PDF เพื่อประกอบการเลื่อนขั้นเงินเดือน
* ⚙️ **ระบบผู้ดูแลระบบ (Admin Panel)**
    * จัดการสิทธิ์การใช้งาน (Roles) และการมอบหมายสิทธิ์รักษาการแทน (Delegation)
    * ระบบซิงค์วันหยุดราชการไทยอัตโนมัติ
    * บันทึกประวัติการใช้งานระบบ (Audit Logs)

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

* **Backend & Database:** Google Apps Script (GAS), Google Sheets
* **Frontend:** Vue.js 3 (Composition API), Vuetify 3 (Material Design Framework)
* **Libraries:** Chart.js (Data Visualization), SweetAlert2 (Popups)
* **Integration:** Google Calendar API, Telegram Bot API / LINE Notify (แจ้งเตือน)

## 🚀 การติดตั้งและใช้งาน (Deployment)

โปรเจกต์นี้ทำงานอยู่บนโครงสร้างของ Google Workspace โดยไม่ต้องตั้งค่าเซิร์ฟเวอร์ (Serverless)

1.  **เตรียมฐานข้อมูล:** สร้างไฟล์ Google Sheets เพื่อใช้เป็นฐานข้อมูล
2.  **เพิ่มโค้ด:** เปิดเมนู `ส่วนขยาย (Extensions)` > `Apps Script` และนำไฟล์โค้ดในโปรเจกต์นี้ไปวาง
    * `รหัส.gs` (Backend Logic)
    * `Index.html` (Frontend UI)
    * `PdfTemplate.html` (Template สำหรับสร้างเอกสาร PDF)
3.  **การนำไปใช้งาน (Deploy):**
    * กดปุ่ม `การทำให้ใช้งานได้ (Deploy)` > `การทำให้ใช้งานได้รายการใหม่`
    * เลือกประเภท: **เว็บแอป (Web App)**
    * เรียกใช้ในฐานะ: **ฉัน (Me)**
    * ผู้มีสิทธิ์เข้าถึง: **ทุกคน (Anyone)**
4.  **ตั้งค่าทริกเกอร์ (Triggers):** * ตั้งเวลาให้ระบบรันฟังก์ชัน `checkVehicleDueDatesAlert` ทุกวันเวลา 08:00 น. เพื่อแจ้งเตือนคิวรถและควันดำ

## 👥 ระดับสิทธิ์ผู้ใช้งาน (User Roles)

ระบบรองรับการแบ่งสิทธิ์ผู้ใช้งานดังนี้:
* **User:** พนักงานทั่วไป (ยื่นใบลา, จองห้องประชุม)
* **Approve 1:** หัวหน้าส่วนราชการ
* **Approve 2:** ปลัด อบต.
* **Approve 3:** นายก อบต. (อนุมัติขั้นสุดท้าย)
* **HR:** ฝ่ายบุคคล (ดูรายงานสถิติทั้งหมด)
* **RoomApprove:** ผู้ดูแลอาคารสถานที่
* **Admin:** ผู้ดูแลระบบ (จัดการตั้งค่าทั้งหมด)

## 🌐 การโฮสต์บน GitHub Pages (Iframe)
